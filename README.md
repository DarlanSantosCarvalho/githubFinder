# GitHub Explorer

Aplicação que consome a API pública do GitHub para buscar e exibir perfis de usuários.

## Funcionalidades planejadas

- Busca de usuários do GitHub por nome de usuário
- Tela de detalhes do repositório com link para o GitHub
- Roteamento client-side via hash (`#/`, `#/user/:username`, `#/user/:username/repo/:repo`)
- Layout responsivo

## Como executar localmente

Utilize a extensão **Live Server** no VS Code e rode a aplicação.
